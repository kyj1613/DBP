index.jsp 실행


멘토 cc(아이디:cc 비밀번호:11)


멘토가 아닌 일반인 mentee01(아이디:mentee01 비밀번호:11) 새로 생성


cc로 로그인을 하면 메인화면에 메뉴와 함께 cc가 회원가입 시 입력한 관심분야에 따라 멘토 추천 리스트가 뜬다. 멘토 추천 리스트는 별점 순으로 정렬된다.

메인 화면에서 Team에 커서를 올려 나타나는 메뉴 중 팀 개설하기를 누른다.

팀이름으로 FinalTeamCC 를 입력, 활동분야 중 하나를 골라 선택하고, 만나는 위치는 서울 같은 지명을 입력, 종료날짜는 'yyyy/mm/dd'형식으로 입력한다.

팀 등록을 누르면 팀 리스트로 이동하며, 팀 리스트에서 모집중인 팀을 누르면 멘티 인원 4명 중 3명 이상이 아닌 팀이 뜨며, 모집 완료된 팀을 누르면 멘티가 3명 이상인 팀이 뜬다.

전체 팀 리스트에서 teamcc3 을 누르면 멘티 인원이 모두 차 qwe(아이디: qwe 비밀번호: 11)으로 접속해서 teamcc3의 세부사항을 보면 신청하는 버튼이 사라져있다.

DOUM이라는 로고를 눌러 메인페이지로 돌아와 메뉴 Team의 특강 리스트 개설하기를 누르면 특강 개설 페이지로 넘어간다. 팀이름으로 finalLargeTeam01을 입력하고 시작날짜를 'yyyy/mm/dd'형식으로 입력 후 활동분야를 선택하고 만나는 위치를 입력한 뒤 모집하고 싶은 특강 수강생 인원을 입력하게되면 그 인원만큼만 수강생을 받을 수 있게 된다. 

특강 등록을 누르면 특강리스트로 돌아오게 되며 현재 모집중인 팀과, 모집 완료된 팀을 선택해 볼 수 있다.



전체 팀 리스트에서 대기명단 중 '씨디이'라는 이름의 checkbox를 체크하고 승인 버튼을 누르면 대기명단에서 '씨디이'가 사라지고 멘티에 '씨디이'가 포함된 모습을 볼 수 있다.

DOUM이라는 로고를 누르면 메인페이지로 다시 이동할 수 있으며 메뉴 중 장소대여 신청 버튼을 누르면 장소 리스트가 뜨고 그 중 하나인 '강남 1호점'을 누르면 '강남 1호점'의 세부사항과 현재 대여내역을 볼 수 있다.

장소대여 신청을 누르면 대여 장소 지점 이름은 자동으로 채워지고 빌리려는 장소 넘버를 선택하고, 장소 대여 등록할 팀의 선택 버튼을 누르면 현재 자신이 멘토로 있는 팀 리스트가 뜨며 그 중 하나를 선택하고, 날짜 입력 칸에는 'YYYY/MM/DD' 형식으로 빌리려는 날짜 이름을 선택하며, 시간입력 칸에서 시간을 선택하고 장소대여 신청을 누른다.

장소대여 신청을 누르면 장소 리스트 화면으로 돌아가고 '강남 1호점'을 선택하면 자신이 선택한 팀이름, 날짜, 시간으로 대여내역에 포함된 것을 볼 수 있다.

오른쪽 상단의 사람 그림을 클릭하면 마이페이지로 이동하며 본인의 상세정보를 볼 수 있고, 자신이 속한 팀 멘티, 팀 멘토, 멘티 승인 대기명단, 특강 참여 내역, 특강 멘토 활동 내역 리스트를 볼 수 있다.

팀 멘토 리스트 중 FinalTeamCC를 클릭하면 FinalTeamCC의 상세사항을 볼 수 있고, 멘토만이 할 수 있는 팀 삭제, 멘티가 직접 하는 멘티 팀 탈퇴, 보고서 리스트, 보고서 리스트 등록로 갈 수 있는 링크가 나와있다. 멘토, 멘티 구분 없이 처음으로 보고서를 등록하면 지원금이 50만원이 지급된다.

보고서와 영수증으로 업로드할 파일을 선택하고, 사용 금액을 입력하면 남은 지원금이 감소되어 나타난다. 보고서 리스트를 누르면 보고서 리스트를 볼 수 있다. 이전페이지로 돌아가 팀 삭제를 누르면 팀에 딸린 보고서와, 멘티, 대기자 명단, 장소대여 내역, 그리고 팀이 삭제된다.





DOUM이라는 로고를 눌러 메인페이지로 돌아와 후기게시판을 누르면 후기게시판 리스트가 출력된다.

글쓰기를 눌러 제목을 입력하고 자신이 멘티 혹은 특강의 수강생으로 있었던, 팀&&특강의 멘토 ID 리스트가 떠 멘토ID를 그 중에서 고를 수 있으며 후기 내용, 형식이 'YYYY/MM/DD'인 후기 날짜를 입력하고 만족성, 참여성, 준비성을 선택해 멘토에게 별을 부여하고 리뷰를 등록한다.

리뷰를 등록하면 후기게시판 리스트로 돌아오고 제목을 누르면 조회수가 올라감과 동시에 리뷰의 세부사항을 볼 수 있다.

오른쪽 상단의 로그아웃을 누르면 로그아웃되어 로그인창으로 돌아온다.

회원가입을 눌러 아이디 mentee01을 입력하고 비밀번호, 이름, 전화번호, 성별, 'yymmdd' 형식의 생년월일, 이메일, 관심분야123을 겹치지 않게 선택 후 회원가입 버튼을 누른다.

로그인창에서 mentee01과 비밀번호를 입력하면 로그인된 메인페이지로 넘어온다. 아직 멘토가 아니기 때문에 팀, 특강을 개설하는 페이지가 나타나지 않으며, Mentor 위에 커서를 올려 멘토 리스트를 누른다.

전체 멘토 리스트가 출력되며, 이름을 누르면 멘토의 세부사항이 출력되며, 멘토 목록으로 돌아가기 버튼을 눌러 멘토 리스트로 돌아온다. 상단의 멘토 목록 검색의 활동분야 중 하나를 누르면 활동분야에 따른 멘토 검색 결과가 화면에 포함된다.

DOUM이라는 로고를 눌러 메인페이지로 돌아와 메뉴 Mentor의 멘토 등록하기를 누르면 멘토 등록 페이지로 넘어간다. 활동분야를 선택하고 자기소개를 입력 후 등록을 누르면 멘토로 등록되며, 멘토리스트로 돌아간다.

DOUM이라는 로고를 눌러 메인페이지로 돌아와 메뉴 Team의 팀 리스트 보기를 누르면 팀리스트가 뜨며, 그 중 하나를 골라 가입 신청하기를 누르면 자신의 이름이 대기명단에 포함된 것을 볼 수 있다. 이후 해당 팀의 멘토가 대기 승인을 하면 팀에 멘티로 등록되며 마이페이지-해당 팀의 이름을 누르면 활동을 시작할 수 있게 된다. 

해당 팀에서 탈퇴하고 싶다면, 마이페이지-멘토링 팀 멘티 활동내역- 해당 팀으로 들어가서 팀 탈퇴 버튼을 누르면 팀에서 탈퇴되며 팀리스트로 돌아온다.

DOUM이라는 로고를 눌러 메인페이지로 돌아와 메뉴 Team의 특강 리스트 보기를 누르면 특강 리스트가 출력되며 팀이름을 누르면 상세정보가 출력된다.

특강 신청하기 버튼을 누르면 현재 인원이 1이 증가됨과 동시에 특강 신청하기 버튼이 사라지며 해당 특강에 포함되게 된다.
